# QR-Boost: Интерфейс игрового приложения

QR-Boost — это интерактивное игровое приложение с поддержкой QR-кодов, позволяющее игрокам погрузиться в мир альтернативной реальности. Проект разработан на React с использованием Convex для серверной части.

## Структура проекта

Проект следует архитектуре Feature-Sliced Design (FSD), состоящей из следующих слоев:

- **pages** — страницы приложения, точки входа для маршрутов
- **widgets** — сложные композиционные компоненты (блоки страниц)
- **features** — части пользовательского интерфейса, отвечающие за функциональность
- **entities** — бизнес-сущности
- **shared** — переиспользуемые компоненты, утилиты, типы

## Разработанные страницы

### 1. QuestLogPage (Журнал заданий)
Страница для отслеживания прогресса по квестам с возможностями:
- Фильтрация квестов по статусу (все, активные, завершенные, проваленные)
- Фильтрация по региону
- Сортировка по разным параметрам
- Подробная информация о выбранном квесте

### 2. InventoryPage (Инвентарь)
Управление инвентарем персонажа:
- Сетка предметов с возможностью сортировки и фильтрации
- Слоты экипировки с визуальным отображением на силуэте персонажа
- Детальная информация о выбранном предмете
- Панель быстрого доступа

### 3. CharacterStatsPage (Характеристики персонажа)
Отображение и управление характеристиками персонажа:
- Основные атрибуты (сила, ловкость, интеллект и др.)
- Вторичные характеристики (здоровье, энергия, урон, защита)
- Прогресс опыта и уровни
- Система распределения очков навыков

### 4. SettingsPage (Настройки)
Настройки приложения:
- Регулировка громкости звуков и музыки
- Выбор языка интерфейса
- Управление аккаунтом
- Раздел помощи и поддержки

### 5. NotFoundPage (Страница 404)
Страница-заглушка при переходе по несуществующему маршруту:
- Информативное сообщение об ошибке
- Кнопки для возврата на предыдущую страницу или на главную

## Технологические особенности

- **Frontend**: React, TypeScript, CSS Modules
- **Backend**: Convex для управления данными
- **QR-функциональность**: Интеграция с камерой для сканирования QR-кодов
- **Адаптивность**: Все компоненты адаптированы для различных устройств
- **Единый стиль**: Консистентный дизайн-система для всех элементов интерфейса
