# Focus Rails

This vault is optimized for two creator tracks:

- Scenario track (`10_Narrative`)
- Game Design track (`20_Game_Design`)

## Operating rule

- At any moment, work in one active track.
- Secondary track can have notes, but no execution until the active track reaches a checkpoint.

## Checkpoints

- Scenario checkpoint:
- scene note updated
- linked quest and location verified
- next scene decided
- Game Design checkpoint:
- intent note updated
- mechanic rule or formula decided
- implementation gap written

## WIP limits

- Active scenarios in parallel: max 2
- Active system mechanics in parallel: max 2
- Open unresolved decisions in one note: max 3

## Priority order

1. Blockers for narrative flow
2. Core mechanic clarity
3. Content expansion
4. Polish
