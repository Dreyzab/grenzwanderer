# Naming Convention

## Canon IDs

- Scenario: `scn_{case}_{name}`
- Scene: `scene_{id}`
- Location: `loc_{name}`
- Quest: `qst_{id}`
- Character: `char_{name}`

## Rules

- Use lowercase snake_case only.
- One entity = one canonical `id`.
- Put legacy names into frontmatter `aliases`.
