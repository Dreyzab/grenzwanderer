# 🔎 Pickelhaube (Spiked Helmet)

**Category**: `Fact`  
**Source**: [[scene_hbf_arrival_beat2]] (Visual)

## Description

Островерхая каска (Pickelhaube) — символ прусского/немецкого полицейского.
Кожаная основа с металлическим шипом наверху.

## Details

| Element   | Description                    |
| --------- | ------------------------------ |
| Material  | Чёрная кожа, латунный шип      |
| Badge     | Кокарда Бадена (красно-жёлтая) |
| Condition | Начищена до блеска             |

## Significance

- Мгновенно идентифицирует носителя как полицейского
- Шип = боевое происхождение (армия → полиция)
